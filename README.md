# BudgetWise - "Your Money, Your Finances"


A personal finance Android app built to help anyone track their transactions, manage budgets, set financial goals, and reduce debt -while it's all stored locally on your device.

## Features

### Transactions
- Add, view and delete Income, Expense and Savings entries
- Attach a photo to any expense
- Filter transactions by date range

### Budgeting
- Set budget goals for each category
- Set a minimum and maximum monthly spending goal
- View category spending totals for any chosen month

### Goals & Debt
- Create and track financial goals
- Record debts and log payments

### Tips & Education
- Browse a built-in list of financial tips

### General
- Offline account - all data is saved locally using RoomDB 
- Automated CI via GitHub Actions

---

## Tech Stack

- Language - Kotlin and java  
- Database - RoomDB
- UI - Material You (Material Design 3) "https://source.android.com/docs/core/display/material"
- Navigation - Bottom and Sidebar navigation
- Testing - using JUnit
- CI - GitHub Actions

## Getting Started on using the Repo

1. Clone the repo
2. Open in Android Studio
3. Let Gradle sync
4. Run on an emulator or on an Android 11 device or above (api 30)
